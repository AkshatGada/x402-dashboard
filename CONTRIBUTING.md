# Contributing to x402 Dashboard

Thank you for your interest in contributing to the x402 Dashboard! We welcome contributions from the community.

## How to Contribute

1. **Fork the repository** on GitHub
2. **Create a feature branch** (`git checkout -b feature/amazing-feature`)
3. **Commit your changes** with clear messages (`git commit -m 'Add amazing feature'`)
4. **Push to the branch** (`git push origin feature/amazing-feature`)
5. **Open a Pull Request**

## Development Setup

```bash
# Clone and install
git clone https://github.com/AkshatGada/x402-dashboard.git
cd x402-dashboard
bun install

# Start development
bun run dev
```

## Commit Message Guidelines

- Use clear, descriptive messages
- Start with a verb: Add, Fix, Update, Remove, etc.
- Keep subject line under 50 characters
- Reference issues when relevant

## Code Style

- Use TypeScript for all code
- Follow Svelte component conventions
- Use Tailwind CSS for styling
- Keep components small and focused

## Testing

Before submitting:
- Test in both light and dark modes
- Test on mobile devices
- Verify API responses in Swagger docs
